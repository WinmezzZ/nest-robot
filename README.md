## Module(现有功能)

- Novel - (随机小说)
- TextCodQuery - (文字编码查询)
- Translation - (同步翻译)
- Weather - (智能天气)

## Installation(安装依赖)

```bash
$ npm install
```

## Running the app(运行应用)

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
